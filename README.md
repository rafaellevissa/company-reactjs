# REACTJS CHALLENGE

## TECHNOLOGIES
- React
- TypeScript
- Material UI (MUI)

## REQUIREMENTS
- NPM

## INSTALL

Once you have the project on your computer, you just need to install the dependencies with npm:

```
npm install 
```

After everything is installed, change the credencials at `.env` and run it with the folowing command:

```
npm start
```
